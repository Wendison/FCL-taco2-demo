Method comparison
  • GT: Ground-truth speech
  • GT(Mel+PWG): Speech synthesized by GT mel-spectrograms with Parallel-WaveGAN vocoder
  • Tacotron2: a strong autoregressive TTS model
  • Fastspeech2: a strong non-autoregressive TTS model
  • FCL-taco2-T: Proposed teacher FCL-taco2 model of similar size to Tacotron2
  • FCL-taco2-S: Proposed student FCL-taco2 model with much smaller size.

Prosody manipulation:

Pitch manipulation: use the predicted F0 multiplied with a ratio (r) to generate the speech

  • r=1: F0 <- F0
  • r=0.5: F0 <- F0 x 0.5
  • r=0.75: F0 <- F0 x 0.75
  • r=1.5: F0 <- F0 x 1.5
  • r=1.25: F0 <- F0 x 1.25
  • r=↗: r linearly increases from 0.5 to 1.5 phoneme by phoneme
  • r=↘: r linearly decreases from 1.5 to 0.5 phoneme by phoneme
